# chn-on-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面为静态网页，内容涵盖不同主题，仅用于信息展示与存档目的。

## 项目简介

`chn-on-guide` 是一个静态 HTML 页面归档仓库。仓库中的每个页面均为独立的 HTML 文件，可直接在浏览器中打开查看。项目不针对任何特定域名或网站，仅作为页面内容的集中存储与发布平台。

## 目录结构

```
chn-on-guide/
├── index.html          # 仓库首页 / 页面索引
├── page1.html          # 示例页面 1
├── page2.html          # 示例页面 2
└── ...
```

- `index.html` 为仓库入口页面，提供对归档页面的导航或说明。
- 其他 `.html` 文件为独立的归档页面，文件名可反映其内容主题。

## 页面归档说明

- 每个 HTML 页面均为完整、独立的静态文档，无需外部依赖即可运行。
- 页面内容可能包含文本、图片、样式和脚本，全部内嵌或引用自安全来源。
- 归档页面可能不定期更新、替换或移除，恕不另行通知。

## 使用方式

1. 直接点击仓库中的 `.html` 文件，在 GitHub 页面中查看渲染效果。
2. 克隆或下载本仓库，在本地浏览器中打开任意 HTML 文件。
3. 若通过 GitHub Pages 部署，可通过对应 URL 访问页面（需自行配置）。
